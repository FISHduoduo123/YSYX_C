# C语言数据类型、运算符及控制结构

## 1. 标准数据类型

### 基本类型

- **int**  
  储存普通的整数，默认为32位大小。  

- **double**  
  储存稍大的浮点数。

- **float**  
  储存稍小的浮点数。

- **char**  
  储存单字节字符。

- **void**  
  表示“无类型”，用于声明不返回任何东西的函数，或者所指类型不明的指针，例如`void *thing`。

- **enum**  
  枚举类型，类似于整数，也可转换为整数，但是通过符号化的名称访问或设置。当`switch`语句中没有覆盖到所有枚举元素时，一些编译器会发出警告。

### 类型修饰符

- **unsigned**  
  修改类型，使它不包含任何负数，同时上界变高。

- **signed**  
  可以储存正数和负数，但是上界会变为（大约）一半，下界变为和上界（大约）等长。  
  *译者注：符号修饰符只对`char`和整数类型有效。`int`默认为`signed`，而`char`根据具体实现，可以默认为`signed`或`unsigned`。*

- **long**  
  对该类型使用较大的空间，使它能存下更大的数，通常使当前大小加倍。

- **short**  
  对该类型使用较小的空间，使它储存能力变小，但是占据空间也变成一半。

### 类型限定符

- **const**  
  表示变量在初始化后不能改变。

- **volatile**  
  表示会做最坏的打算，编译器不会对它做任何优化。通常仅在对变量做一些奇怪的事情时，才会用到它。

- **register**  
  强制让编译器将这个变量保存在寄存器中，并且也可以无视它。目前的编译器更善于处理在哪里存放变量，所以应该只在确定这样会提升性能时使用它。

### 类型转换

C使用了一种“阶梯形类型提升”的机制：观察运算符两边的变量，在运算之前将较小边的变量转换为较大边。转换顺序如下：
long double → double → float → long long → long → int (short, char)

## 2. stdint.h 库类型

`stdint.h`为定长的整数类型定义了一些`typedef`，同时也有一些用于这些类型的宏。这比老的`limits.h`更加易于使用，因为其大小是不变的。

### 定长整数类型

| 类型      | 描述             | 最大值宏         | 最小值宏         |
|-----------|-----------------|----------------|-----------------|
| `int8_t`  | 8位符号整数      | `INT8_MAX`      | `INT8_MIN`      |
| `uint8_t` | 8位无符号整数    | `UINT8_MAX`     | 0（隐含）         |
| `int16_t` | 16位符号整数     | `INT16_MAX`     | `INT16_MIN`     |
| `uint16_t`| 16位无符号整数   | `UINT16_MAX`    | 0               |
| `int32_t` | 32位符号整数     | `INT32_MAX`     | `INT32_MIN`     |
| `uint32_t`| 32位无符号整数   | `UINT32_MAX`    | 0               |
| `int64_t` | 64位符号整数     | `INT64_MAX`     | `INT64_MIN`     |
| `uint64_t`| 64位无符号整数   | `UINT64_MAX`    | 0               |

*译者注：当用于对类型大小有要求的特定平台时，可以使用这些类型。如果你怕麻烦，不想处理平台相关类型的潜在扩展，也可以使用它们。*

> **警告**：不要从字面上在任何头文件中去找`INT(N)_MAX`的定义。这里的`N`应该为特定整数，比如8、16、32、64，甚至可能是128。本文使用记法`INT(N)_MAX`是为了简洁。

### 至少 N 位类型

保证至少`N`位，实际可能更大。

| 类型                  | 最大值宏               | 最小值宏               |
|-----------------------|------------------------|------------------------|
| `int_least(N)_t`      | `INT_LEAST(N)_MAX`     | `INT_LEAST(N)_MIN`     |
| `uint_least(N)_t`     | `UINT_LEAST(N)_MAX`    | 0                      |

### 最快类型

至少`N`位的“最快”整数（可能占用更多内存但访问速度快）。

| 类型                | 最大值宏             | 最小值宏             |
|---------------------|----------------------|----------------------|
| `int_fast(N)_t`     | `INT_FAST(N)_MAX`    | `INT_FAST(N)_MIN`    |
| `uint_fast(N)_t`    | `UINT_FAST(N)_MAX`   | 0                    |

### 指针相关类型

| 类型         | 描述                         | 最大值宏         | 最小值宏         |
|--------------|------------------------------|------------------|------------------|
| `intptr_t`   | 足够存放指针的符号整数       | `INTPTR_MAX`     | `INTPTR_MIN`     |
| `uintptr_t`  | 足够存放指针的无符号整数     | `UINTPTR_MAX`    | 0                |

### 最大尺寸整数类型

系统中可能的最大尺寸的整数类型。

| 类型          | 最大值宏         | 最小值宏         |
|---------------|------------------|------------------|
| `intmax_t`    | `INTMAX_MAX`     | `INTMAX_MIN`     |
| `uintmax_t`   | `UINTMAX_MAX`    | 0                |

### 其他常用宏

- `PTRDIFF_MIN` / `PTRDIFF_MAX`：`ptrdiff_t`的最小/最大值。
- `SIZE_MAX`：`size_t`的最大值。

## 3. 运算符及补充语句

### 运算符分类

#### 算数运算符

| 运算符 | 类型       | 说明                                         |
|--------|------------|----------------------------------------------|
| `()`   | 函数调用   | -                                            |
| `*`    | 二元乘法   | -                                            |
| `/`    | 除法       | -                                            |
| `+`    | 一元正号 / 二元加法 | 一元无变化；二元求和                    |
| `-`    | 一元负号 / 二元减法 | 一元取反；二元求差                    |
| `++`   | 自增       | 前缀：先自增后读取；后缀：先读取后自增       |
| `--`   | 自减       | 前缀：先自减后读取；后缀：先读取后自减       |

#### 数据运算

| 运算符 | 说明                                                   |
|--------|--------------------------------------------------------|
| `->`   | 结构体指针的成员访问（复合`*`和`.`）                   |
| `.`    | 结构体值的成员访问                                     |
| `[]`   | 取数组下标（复合`+`和`*`）                             |
| `sizeof` | 取类型或变量大小（字节）                             |
| `&`    | 取地址（一元）                                         |
| `*`    | 取值（提领地址，一元）                                 |

#### 逻辑运算符

| 运算符 | 说明             | 运算符 | 说明         |
|--------|------------------|--------|--------------|
| `==`   | 等于             | `!=`   | 不等于       |
| `<`    | 小于             | `<=`   | 小于等于     |
| `>`    | 大于             | `>=`   | 大于等于     |
| `!`    | 逻辑非（取反）   |        |              |

#### 位运算符

| 运算符 | 说明         |
|--------|--------------|
| `&`    | 位与（二元） |
| `\|`   | 位或（二元） |
| `^`    | 位异或（二元）|
| `~`    | 取补（一元，翻转所有位） |
| `<<`   | 左移         |
| `>>`   | 右移         |

#### 布尔运算符

| 运算符 | 说明                 |
|--------|----------------------|

| `\|\|` | 逻辑或               |
| `?:`   | 三元运算符：`X ? Y : Z` 读作“若X则Y否则Z” |

#### 赋值运算符

| 运算符 | 说明         | 运算符 | 说明           |
|--------|--------------|--------|----------------|
| `=`    | 赋值         | `%=`   | 取余赋值       |
| `*=`   | 乘法赋值     | `&=`   | 位与赋值       |
